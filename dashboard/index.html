Setup
1

Install Node.js

FokkerPop runs on Node.js 18 or later. If you're reading this, Node is already running. ✅

2

Connect Twitch

Create a Twitch app at dev.twitch.tv/console. Set the OAuth Redirect URL to:
http://localhost:4747/auth/callback

3

Add Browser Source in OBS

In OBS, add a Browser Source to your scene with these settings:

http://localhost:4747/

Width: 1920  ·  Height: 1080  ·  ✅ Shutdown source when not visible  ·  ✅ Custom CSS: body { background: transparent !important; }

4

Start Streaming

Use start.bat to launch FokkerPop before going live. The dashboard opens automatically. Check the Live tab to confirm Twitch events are flowing in.

Tip: add start.bat to your stream startup shortcut so it launches alongside OBS.
