Configure Goals●

Create targets like 100 subs or 50,000 bits. When a goal is reached, the chosen reward effect fires automatically. Metric examples: session.subCount, session.bitsTotal, session.followCount, session.raidCount.

Overlay Layout
Drag Mode

Turn Drag Mode on, then click-and-drag any widget in the preview below to reposition it. Every widget appears with a dashed purple outline even if it has no data yet (Leaderboard before any bits, Timer before any countdown, Goals if none configured), so you can place them first and let data fill in later. Positions save automatically. Turn Drag Mode off when you're done.

Tip: dragging here, in the Pop-out Editor, and using OBS → right-click source → Interact all write to the same saved positions.

Preview (preview-only, OBS overlay always shows widget labels)
Custom Widgets

Custom overlay widgets save to widgets.json and appear in every overlay (Test Effects preview, Pop-out Editor, OBS). Drag in the preview above with Drag Mode on to position them. Counter binds to a metric path like session.subCount or leaderboard.bits.TopUser. Text is a static label. Hot Button fires a chosen effect when clicked in OBS (via right-click → Interact) or in the preview.

Config Editor
Auto-Install Updates

When on, FokkerPop will check GitHub every 30 minutes and automatically apply new releases. Your settings, goals, redeems, commands, flows, and uploaded assets are preserved — only the app code and default samples are overwritten. If you're actively streaming, install waits until the stream ends.

You can always trigger a one-time install from the gold banner when an update is detected, even with this off.

Layout Mode

When active, you can click and drag HUD elements (Crowd Meter, Goals, etc.) directly in your OBS browser source or preview window to reposition them.

Pro Tip: To move things directly in OBS, right-click your Browser Source and select Interact.

Import from PolyPop

        

          Migrating from PolyPop? Pick your .pop project file — FokkerPop will pull
          out your channel-point redeems and chat-command aliases. Only the 1:1 pieces (redeem
          titles, audio file references) come over; PolyPop's Action Sequences, scenes, and 3D
          models don't have FokkerPop equivalents and are skipped.
        

        

          
          
          
        

        

      


      

        
Shut down FokkerPop

        

          Stops the server cleanly — broadcasts a shutdown signal to every connected overlay (so OBS browser sources clear their effect queues), flushes state.json, then exits. You'll need to run start.bat (or the FokkerPop Start Menu shortcut, or the desktop FokkerPop.exe) to bring it back up.
