public-domain. Credit requirements (for CC-BY only) are listed below.

- **License:** [Creative Commons Attribution 3.0 (CC-BY 3.0)](https://creativecommons.org/licenses/by/3.0/)
- **Originals:**
  - `dice1.wav` — [Roll Dice 2](https://soundbible.com/181-Roll-Dice-2.html)
  - `dice2.wav` — [Shake And Roll Dice](https://soundbible.com/182-Shake-And-Roll-Dice.html)

If you rebroadcast or distribute FokkerPop output in a form where the licensor
must be credited (typical streaming use falls under fair use), keep this file
in the release so the attribution travels with the sounds.

## Other bundled sounds

The remaining `.wav` files in this directory (`coin`, `laser`, `woosh`, and
any others) ship with FokkerPop as default effect SFX. Replace any of them at
will — FokkerPop picks them up by filename, and missing files fall back to
`alert.wav`.
